Directives

Directive são uma instrução inseridas dentro do DOM.

Components são casos específicos de uma Directive.

Podemos usar Directives existentes ou construir as nossas.

Uma Directive é uma classe com o decorator @Directive

Existem outros dois tipos de Directive.

1 - Structural DIrective - Alteram o layout da DOM, adicionando, removendo ou substituindo elementos.

2 - Attribute Directive - Alteram o aspecto e comportamento de elementos existentes.
